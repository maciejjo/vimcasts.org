I'll be running a Vim Masterclass in Boulder, Colorado on September 19th, as part of the Rocky Mountain Ruby Charity Workshop day. This is a rare USA appearance, so don't miss out! Donations will go to local charities in Boulder. Please give generously.

Read about what to expect from a Vim Masterclass if you'd like to know more about the format of my workshops.

You can register for the Vim masterclass on eventbrite. There are 6 other workshops taking place on the same day, so be sure to check those out too:

  • RailsBridge: Intro to Ruby and Rails
  • Playing with Ruby
  • Front-end architecture with Sass 3.2
  • The Hashrocket way
  • First Steps testing JavaScript with Jasmine
  • Ruby meta programming

Please give generously.

The workshops are open to anyone, not just those who are attending the Rocky Mountain Ruby conference. I hope to see you there.
